# SqlWolf

SqlWolf is a cross-platform SQL editor and database manager available for Linux, Mac, and Windows.

> This project is a fork of [Beekeeper Studio](https://github.com/beekeeper-studio/beekeeper-studio).

## Features

- Truly cross-platform: Windows, macOS, and Linux
- Autocomplete SQL query editor with syntax highlighting
- Tabbed interface for multitasking
- Sort and filter table data
- Sensible keyboard shortcuts
- Save queries for later
- Query run history
- Dark theme
- Import/export
- Backup/restore
- Azure Key Vault integration for credential management
- View data as JSON
